# IMPORTANTE
Por razones desconocidas, mi API KEY de TMDB no me funcionó. En su lugar utilicé el API Read Access Token / Authorization Token.
El programa está diseñado para aceptar solo el Token, no funcionará si se utiliza la Api Key.

El Token se genera cuando se crea una cuenta de TMDB, al igual que la Api Key.
El Token se debe insertar en la constante API_KEY que se encuentra en el archivo getData.js, en la carpeta "utils" dentro de "src."
